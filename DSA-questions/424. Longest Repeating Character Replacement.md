# 424. Longest Repeating Character Replacement

#### [Leetcode](https://leetcode.com/problems/longest-repeating-character-replacement/)

## Approach - 1 
```java
// notes : 
// shrink window when the no of non freq occurring elements in window are more than k 
// because we can only replace k non freq chars only so this is not a valid window so shrink untill
// out window has no of non frequently occ chars == k
// windowsize - maxCount => no of non freq chars that needs to be replaced in the window to make all chars same
            
class Solution {
    public int characterReplacement(String s, int k) {
        int[] count = new int[26]; // to store freq of chars in window
        int maxLen = 0;
        int maxCount = 0; // stores count of most freq occurring ele in window
        int left = 0;
        for(int right=0; right < s.length(); right++){

            count[s.charAt(right) - 'A']++;
            maxCount = Math.max(maxCount, count[s.charAt(right) - 'A']);

            // shrink window when the no of non freq occurring elements in window are more than k 
            while(right - left + 1 - maxCount > k){
                count[s.charAt(left) - 'A']--;
                left++;
            }

            maxLen = Math.max(maxLen, right - left + 1);
        }

        return maxLen;
    }
}
```
