# 572. Subtree of Another Tree

### [Leetcode](https://leetcode.com/problems/subtree-of-another-tree/)

### Approach - 1 (using some same tree logic)
```java
// using the sameTree logic 
// prerequisite : sameTree : https://leetcode.com/problems/same-tree/
// time : O(n*n) Space : O(n) - recursive stack

class Solution {
    private boolean isSameTree(TreeNode root, TreeNode subRoot){
        // 
        if(root == null && subRoot == null)
            return true; 
        else if(root == null || subRoot == null)
            return false;
        else if(root.val == subRoot.val) 
            return isSameTree(root.left, subRoot.left) && isSameTree(root.right, subRoot.right);
        else return false; // can't be same
    }

    public boolean isSubtree(TreeNode root, TreeNode subRoot) {
        
        if(root == null && subRoot == null) // both empty, same
            return true; 
        else if(root == null || subRoot == null) // 1 empty, not same
            return false;
        
        if(isSameTree(root, subRoot)) // check if curr tree are same
            return true;

        // find subtree in left of main tree Or right of main tree
        else return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot);
    }
    
}
```
