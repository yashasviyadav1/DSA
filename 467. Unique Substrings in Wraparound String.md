# 467. Unique Substrings in Wraparound String

#### [Leetcode](https://leetcode.com/problems/unique-substrings-in-wraparound-string/)

## Approach - 1 (using extra space array)
```java
// idea : in 1 pass we find and store the longest contigious substring ending with a char, and we store it for all char in count array and at the end we sum the substrings
// the sole reason for storing the long cont substri ending with a char is that if need unique substrings and no duplicate substrings so
//lets say we have 2 contigious substrings ending with c as : abc and zabc then we choose the longest one and count[c] = 4

class Solution {
    public int findSubstringInWraproundString(String s) {
        int[] count = new int[26]; // storing len of longest contigious substring ending for each characters a to z

        int maxLen = 0;
        for(int i=0; i < s.length(); i++){

            // if curr char is contiguous to prev char of string then increment the len, if not then reset the maxLen 
            if(i > 0 && ( (s.charAt(i-1) + 1 == s.charAt(i)) || (s.charAt(i-1) - s.charAt(i) == 25) )) 
                maxLen++;
            else maxLen = 1; // 

            char ch = s.charAt(i);
            count[ch - 'a'] = Math.max(count[ch - 'a'], maxLen);
        }

        int sum = 0;
        for(int i=0; i < 26; i++)
            sum += count[i];

        return sum;

    }
}
```
